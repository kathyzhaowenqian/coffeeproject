前言：

本报告以展示编程技术对于市场调研及业务分析的辅助作用为主，图表配套文字仅对数据作简单阐述。所有调研成果的核心依据为数据，通过编程（爬虫技术）可以获取完整的原始数据，对于业务而言具有很高价值。获取数据之后，还可通过编程语言批量清洗、分析数据，并编译成可视化图表，以供后期分析报告的撰写，并同时提高工作效率。本报告所运用到的编程语言、工具库等技术如下：

编程语言： Python

Python由荷兰数学和计算机科学研究学会的吉多·范罗苏姆 于1990 年代初设计，作为一门叫做ABC语言的替代品。 Python提供了高效的高级数据结构，还能简单有效地面向对象编程。

Python语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚本和快速开发应用的编程语言，  随着版本的不断更新和语言新功能的添加，逐渐被用于独立的、大型项目的开发。  Python解释器易于扩展，可以使用C语言或C++（或者其他可以通过C调用的语言）扩展新的功能和数据类型。  Python 也可用于可定制化软件中的扩展程序语言。

Python丰富的标准库，提供了适用于各个主要系统平台的源码或机器码。  2021年10月，语言流行指数的编译器Tiobe将Python加冕为最受欢迎的编程语言，20年来首次将其置于Java、C和JavaScript之上。

HTML

HTML的全称为超文本标记语言，是一种标记语言。它包括一系列标签，通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。 HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等。

CSS

层叠样式表(英文全称：Cascading Style Sheets)是一种用来表现HTML（标准通用标记语言的一个应用）或XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

JavaScript

JavaScript（简称“JS”） 是一种具有函数优先的轻量级，解释型或即时编译型的编程语言。虽然它是作为开发Web页面的脚本语言而出名，但是它也被用到了很多非浏览器环境中，JavaScript 基于原型编程、多范式的动态脚本语言，并且支持面向对象、命令式、声明式、函数式编程范式。


Web开发框架： Django

Django是一个开放源代码的Web应用框架，由Python写成。采用了MTV的框架模式，即模型M，视图V和模版T。 它最初是被开发来用于管理劳伦斯出版集团旗下的一些以新闻内容为主的网站的，即是CMS（内容管理系统）软件。并于2005年7月在BSD许可证下发布。

Vue

Vue是一套用于构建用户界面的渐进式JavaScript框架，作为目前最流行的前端框架之一，是许多前端开发工程师的不二选择。与传统JS和JQuery框架不同，Vue的渐进式框架表示开发者可以由简单组件写起，渐渐搭建出一个复杂的前端平台。


数据获取方式： 网络爬虫

网络爬虫是一种按照一定的规则，自动地抓取万维网信息的程序或者脚本。另外一些不常使用的名字还有蚂蚁、自动索引、模拟程序或者蠕虫。


数据分析调用工具库： Pandas

Pandas 是基于NumPy 的一种工具，该工具是为解决数据分析任务而创建的。Pandas 纳入了大量库和一些标准的数据模型，提供了高效地操作大型数据集所需的工具。Pandas提供了大量能使我们快速便捷地处理数据的函数和方法。你很快就会发现，它是使Python成为强大而高效的数据分析环境的重要因素之一。


数据可视化调用工具库： Pyecharts

Echarts是一个由百度开源的数据可视化，凭借着良好的交互性，精巧的图表设计，得到了众多开发者的认可。 而 Python 是一门富有表达力的语言，很适合用于数据处理。当数据分析遇上数据可视化时，Pyecharts 诞生了。



生成此类报告的工作可拆解为：1. 获取原始数据； 2. 数据整合、清洗和分析； 3. 可视化图表制作； 4. 分析报告撰写； 5. 网页开发。 在专业科技公司通常由多部门团队协作。由于此次是我第一个实战项目，总耗时超过100个小时，其中约50个小时用在数据获取代码的编写及数据获取过程中； 约55个小时用在数据分析和可视化图表代码的编写中；约20个小时用在Web开发框架代码编写中；约15个小时用在报告文字撰写和整体排版中。 由于本报告图表均通过Python语言编写，在视觉呈现上没有做过多修饰，后期可调用专业可视化工具提升视觉效果。

数据获取逻辑：
  • 本报告引用数据均通过爬虫技术从大众点评获取，搜索关键字为“咖啡”（模糊查询），选取全国19个重点城市：上海、北京、广州、成都、深圳、杭州、苏州、重庆、南京、武汉、青岛、天津、宁波、无锡、厦门、昆明、合肥、郑州、济南。 数据信息包括：城市、门店名称、消费均价、星级评分、点评数、地区、推荐菜品等等。通过“咖啡”关键字获取的门店可为咖啡厅，或者推荐菜品中包含“咖啡”的门店。
  • 为尽可能保留原始数据，获取到的除咖啡厅以外的门店，如：茶饮、快餐、餐厅等，均做保留。相比大众点评上的全部门店信息，本数据完整性为95%以上。 从业务层面，包含“咖啡”关键字的非咖啡厅门店会给分析报告的精确度带来一定影响，如高端餐厅会影响均价，连锁快餐厅会影响门店品牌排行。
  • 若作为公司项目，搜索关键字、数据过滤、以及后期的图表类型的选择和数据维度筛选，都需要做进一步细化、探讨。

总结：
  • 通过对大众点评门店源数据的分析，咖啡作为燕麦奶的传统搭配，除了咖啡厅以外，所有售卖咖啡的门店，大到快餐连锁，小到单店餐厅，均有洽谈合作的机会。
  • 燕麦奶品类在近几年已广泛渗透餐饮行业，以至于在推荐菜品（无论是推荐频数还是推荐指数）中都可以看到多款燕麦奶饮品。
  • 植物基产品在中国市场发展还需对消费者进行持续教育，不同燕麦奶品牌可以通过切入渠道的不同，打造差异化竞争。如饮品渠道的开拓不仅局限于咖啡厅，像连锁品牌排行表中出现的茶饮品牌Coco都可；烘焙品牌85度C、味多美；冰激凌品牌哈根达斯；以及快餐、便利店、超市、网红餐厅等，均为可以拓展的方向。
  • 在产品矩阵上，可根据热门饮品品类（推荐菜品）结合进行研发。除了饮品以外，像冰激凌、烘焙、中西式甜品、甚至拓展到高端西餐菜肴的辅料配料等，都是可以探索的方向。结合市场流行趋势研发相关配方，结合双方品牌调性与消费者讲故事。
  • 根据全国不同城市的市场情况，结合当地商圈和连锁品牌的影响力和契合度来布局，打造常规及差异化产品矩阵，实现销量和品牌曝光双增长。
